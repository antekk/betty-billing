 0   1 DEFINITIONS
 0
 0     This document, entitled the Schedule of Medical Benefits is hereinafter
 0     referred to as "Schedule".  This Schedule applies only to those services
 0     that are insured under the Alberta Health Care Insurance Act.  These General
 0     Rules (GRs) apply to all benefits unless otherwise stated.
 0
 0    1.1 In this Schedule, "certificate of registration" means a health insurance
 0        card issued under the Health Insurance Premiums Act to a resident of
 0        Alberta, or any other document prescribed as a "certificate of registration"
 0        for the purposes of the Health Insurance Premiums Act and the Medical
 0        Benefits Regulation.
 0
 0    1.2 "holidays" or "statutory holidays" means New Year's Day, Family Day, Good
 0        Friday, Victoria Day, Canada Day, Alberta Heritage Day, Labour Day,
 0        Thanksgiving Day, Remembrance Day, Christmas Day, Boxing Day;
 0

 0        another day as the holiday;
 0
 0    1.4 "physician" includes "osteopath";
 0

 0        or temporary lodging, assisted living, designated assisted living, group
 0        home, seniors' lodge, personal care home and other residences as approved,
 0        but does not include auxiliary hospitals or nursing homes.

 0        grandparents, spouse or adult interdependent partner or any person who is
 0        dependent on the practitioner for support in accordance with the Alberta

 0   2 APPLICATIONS
 0
 0    2.1 The benefits payable for services provided inside or outside Alberta by or
 0        under the supervision of a physician, shall be the benefit prescribed in the
 0        Schedule subject to GRs outlined herein.
 0

 0    2.2 Where a specific case contradicts a general statement within these GRs the
 0        specific shall override the general statement.
 0
 0    2.3 Claims for unlisted services will be assessed by comparing the service
 0        provided and the fee claimed with similar or comparable services listed in
 0        the Schedule.  The assessment will be based on the time, complexity, and
 0        intensity of the services.  Supporting information, such as an operative
 0        report, is required with the claim.
 0
 0     2.3.1 Unless otherwise specified, services that may be claimed once per year may
 0           be claimed 365 days after the previous service date or 366 days in a leap
 0           year.
 0
 0     2.3.2 Cumulative time is calculated by adding the total time spent delivering
 0           patient care as identified in the description of the HSC, over the course
 0           of the day (GR 1.19) and dividing the total time by the time units specified
 0           in the HSC to determine the appropriate number of calls. When the remainder
 0           of the time calculation equals less than half of one call, an additional
 0           call may not be claimed. Separate encounters may only be claimed when a
 0           special call for attendance has been made on the patient's behalf.
 0
 0     2.3.3 Where time is described as a full amount of minutes e.g. a full 5 minutes,
 0           the physician must spend the full amount of time stated in the HSC in order
 0           to submit a claim for the service.
 0
 0     2.3.4 Where time is described as a portion thereof, the physician may spend any
 0           amount of time providing the services described by the HSC in order to
 0           submit a claim for the service.
 0
 0     2.3.5 Where time is described as a major portion thereof, the physician must spend
 0           a minimum of half of the time described in the HSC providing the service in
 0           order to submit a claim for the service.  Additional calls for the same HSC
 0           may not be claimed until the full time period as described in the HSC for
 0           each previous call has elapsed.
 0
 0     2.3.6 When billing time based services, including modifiers, the physician must
 0           document the time spent providing time based services for each day of
 0           service (as defined in GR 1.19).  The record must be available upon request
 0           and should be kept in chronological order, for each day.  The total time
 0           claimed for time based services in a single day cannot exceed the total time
 0           spent delivering patient care activities in relation to an insured service.
 0           Claims for services that are described as cumulative time, major portion
 0           thereof or portion thereof may continue to be submitted in accordance to
 0           GR's 2.3.2, 2.3.4 and 2.3.5.
 0
 0     2.3.7 Concurrent billing for overlapping time for separate patient encounters/
 0           services may not be claimed.
 0

 0    2.7 CLAIMS FOR BENEFITS
 0
 0     2.7.1 A claim must be submitted in the format prescribed by the Minister.
 0
 0     2.7.2 GR 2.7.1 applies whether the claim is submitted by a physician on behalf
 0           of a patient or by the patient.
 0

 0           midnight.  A hospital visit which takes place after 0700 hours may be
 0           claimed in addition to one of the following services provided between
 0           midnight and 0700 hours:

 0          b)  home visit;

 0          d)  emergency visit/special callback to hospital emergency/outpatient
 0              department, AACC or UCC, when specially called from home or office;
 0          e)  a special call for attendance to a patient at a closed office, with no
 0              staff in attendance.

 0           for benefits is payable subject to the timelines indicated in the Alberta
 0           Health Care Insurance Act and regulations.

 0           resident or intern during the provision of a service.

 0    4.2 VISITS - DEFINITIONS
 0
 0     4.2.1 Brief Visit:  Assessment of a patient's condition when history is minimal
 0           and little or no physical examination is included.
 0
 0     4.2.2 Limited Visit:  A limited assessment, of a patient, which includes a history
 0           limited to and related to the presenting problem, and an examination which
 0           is limited to relevant body systems, an appropriate record, and advice to
 0           the patient.  It includes the ordering of appropriate diagnostic tests and
 0           procedures as well as discussion with the patient.
 0
 0     4.2.3 Comprehensive Visit:  An in-depth evaluation of a patient.  This service
 0           includes the recording of a complete history and performing a complete
 0           physical examination appropriate to the physician's specialty, an
 0           appropriate record and advice to the patient.  It may include the ordering
 0           of appropriate diagnostic tests and procedures as well as discussion with
 0           the patient.
 0           Advice to the patient must include discussion of a care plan related to the
 0           patient's condition(s). Patient care advice, including the discussed care
 0           plan, must be documented in the patient's record. The care plan does not
 0           have to be formally signed by the patient.
 0
 0     4.2.4 Palliative Care: Defined as care given to a patient with a terminal disease
 0           such as cancer, AIDS or advanced neurologic disease. Palliative care
 0           involves active ongoing multi-disciplinary team care. Physicians involved in
 0           palliative care may claim for services provided under 03.05I, 03.05IZ,
 0           03.05T and 03.05U as applicable.
 0
 0     4.2.5 Chronic Pain: Defined as pain which persists past the normal time of
 0           healing, is associated with protracted illness or is a severe symptom of a
 0           recurring condition.
 0           Interdisciplinary Chronic Pain Program: Defined as a comprehensive,
 0           coordinated, interdisciplinary program for persons complaining of chronic
 0           pain. The interdisciplinary team consists of a medical director; other team
 0           members will include psychologist(s) and/or psychiatrist(s),
 0           physiotherapist(s) and/or occupational therapist(s) and may include
 0           anesthetist(s) and other professional personnel. Treatment is delivered by a
 0           coordinated team within the same site by an interdisciplinary chronic pain
 0           program.

 0    4.3 CONSULTATIONS - DEFINITIONS
 0
 0     4.3.1 Comprehensive Consultation: An in-depth evaluation of a patient with a
 0           written report to the referring physician, audiologist, Alberta registered
 0           midwife, chiropractor, podiatrist, dentist, optometrist, physical therapist
 0           or nurse practitioner. This service includes the recording of a complete
 0           history, performing a complete physical examination appropriate to the
 0           physician's specialty, an appropriate record and advice to the patient. It
 0           may include the ordering of appropriate diagnostic tests and procedures as
 0           well as discussion with the patient and/or the referring physician,
 0           audiologist, midwife, chiropractor, podiatrist, dentist, optometrist,
 0           physical therapist or nurse practitioner.
 0
 0     4.3.2 Limited Consultation: Limited assessment of a patient and a written report
 0           to the referring physician, audiologist, Alberta registered midwife,
 0           chiropractor, podiatrist, dentist, optometrist, physical therapist or nurse
 0           practitioner. A limited consultation includes a history limited to and
 0           related to the presenting problem, and an examination which is limited to
 0           relevant body systems, an appropriate record, and advice to the patient. It
 0           may include the ordering of appropriate diagnostic tests and procedures as
 0           well as discussion with the patient and/or the referring physician,
 0           audiologist, midwife, chiropractor, podiatrist, dentist, optometrist,
 0           physical therapist or nurse practitioner.
 0
 0     4.3.3 Time Based Consultations: Notwithstanding GRs 4.3.1 and 4.3.2, claims for
 0           consultation services as defined under HSCs 03.08F, 03.08I, 03.08IV,
 0           03.08IZ, 03.08J, 03.08JV, 03.08JZ, 03.08L, 03.08M, 08.19A, 08.19AZ,
 0           08.19AA, 08.19B, 08.19BB, 08.19C, 08.19CC, and 08.19CX may be claimed on
 0           a time basis.
 0
 0     4.3.4 Psychiatric Consultation referred by other professions: A benefit for a
 0           psychiatric consultation (HSCs 08.19AA, 08.19BB, 08.19CC) may be claimed
 0           when a patient is referred to a psychiatrist by a registered: occupational
 0           therapist, psychologist, community based psychiatric nurse, social worker or
 0           speech language pathologist and the provisions that apply to consultations
 0           under GRs 4.3, 4.4 and 4.6 are met.

 0    4.4 CONSULTATION - APPLICATION
 0
 0     4.4.1 In this Schedule "consultation" means that situation where a physician,
 0           audiologist, midwife, chiropractor, podiatrist, dentist, optometrist,
 0           physical therapist or nurse practitioner after an appropriate examination of
 0           the patient, requests the opinion of a consultant physician, and the
 0           consultant does a history, an examination and a review of the diagnostic
 0           data and provides a written opinion with recommendations as to the
 0           treatment, to the referring physician, audiologist, midwife, chiropractor,
 0           podiatrist, dentist, optometrist, physical therapist or nurse practitioner.
 0           Consultations may not be claimed for the transfer of care alone.
 0
 0     4.4.2 The need for a consultation can arise as a result of the following:
 0          a)  some unusual or serious clinical problem,
 0          b)  a physician, audiologist, midwife, chiropractor, podiatrist, dentist,
 0              optometrist, physical therapist or nurse practitioner requires further
 0              advice regarding diagnosis or management or both, or

 0          c)  the patient, parent or guardian requests another opinion.
 0
 0     4.4.3 A referral may be accepted from any person; however, to receive
 0           reimbursement as a consultation, a request must be made by the referring
 0           physician, audiologist, midwife, chiropractor, podiatrist, dentist,
 0           optometrist, physical therapist or nurse practitioner to the consultant in
 0           the form of:
 0          a)  verbal or written communication (fax, email, letter);
 0          b)  verbal or written communication between an agent representing the referring
 0              physician, audiologist, midwife, chiropractor, podiatrist, dentist,
